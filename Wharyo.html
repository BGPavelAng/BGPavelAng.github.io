En esta prueba son multiples pasos para poder avanzar.

La primera prueba se trata sobre que debe el página detectar que es un browser en particular, nos pide que se llame picoBrowser.

Vamos a utilizar Burp para todo esto y todo lo puedes encontrar en la página de Mozilla -> MDN web docs.

Para el primer cambio, user-agent se utiliza para detectar el tipo de browser que estas utilizando, hagamos un cambio y coloquemos picoBrowser.

Ahora nos dice que no acepta usuarios de otros lugares, esto siginifica que tiene que estar el usuario haciendo la petición desde el mismo servidor, así que utilizaremos Referer para decirle que estamos haciendo la petición desde la misma página.

Referer: http://mercury.picoctf.net:46199/

Ahora la página pide que la fecha sea del 2008.

usemos Date: Date: Wed, 21 Oct 2018 07:28:00 GMT

Olvide tomar captura, pero ahora nos pide que bloquemos el seguimiento.

usemos DNT: 1

Ahora nos pode que seamos de Suecia.

Usemos un proxy de Suecia

X-Forwarded-For: 103.45.246.0

Se siente como si nunca fuera a terminar.

Ahora nos pide que si somos de suiza, hablemos su idioma.

Accept-Language: sv
